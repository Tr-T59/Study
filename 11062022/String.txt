Разработать класс String, который в дальнейшем будет 
использоваться для работы со строками. Класс должен содержать: 
• Конструктор по умолчанию, позволяющий создать 
строку длиной 80 символов; 
• Конструктор, позволяющий создавать строку произвольного размера; 
• Конструктор, который создаёт строку и инициализирует её строкой, полученной от пользователя;
• Добавить в конструктор переноса.
• Добавьте перегрузку [], (), преобразования типа к int:
• [] — возвращает элемент по указанному индексу;
• () — ищет символ в строке, если символ есть возвращает индекс, если нет –1.
Преобразование к int возвращает длину строки.
